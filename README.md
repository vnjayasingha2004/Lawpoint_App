# LawPoint

LawPoint is a legal services platform with:
- client and lawyer accounts
- legal locker
- encrypted messaging
- knowledge hub
- appointment booking
- lawyer verification workflow

## Tech Stack
- Flutter
- Node.js / Express
- PostgreSQL

## Setup

### Backend
1. Edit `.env` put your db name and db password correctly and storage file path to storage dir
2. put your payhere merchant secret and ngrok url on http500 in `.env` 
3. Fill your local values
4. Install packages
5. Run the server

### Database
1. Create PostgreSQL database
2. Run `database/schema.sql`


### Flutter
1. `flutter pub get`
2. run the app
